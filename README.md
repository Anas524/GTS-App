# GTS Application

A modular Laravel-based business operations platform designed to manage financial workflows, documents, and logistics in a single system.

---

## 🚀 Core Modules

### 🧮 Amazon Revenue Calculator
Estimate landed cost, marketplace fees, and net margins in seconds.  
Save and compare scenarios for smarter pricing decisions.

---

### 📊 Investment Sheets
Evaluate opportunities, track ROI, and generate investor-ready reports.  
Supports multi-entry inputs and structured financial analysis.

---

### 📥 Leads Management
Capture and manage contact messages and newsletter signups.  
Centralised lead tracking for better follow-up workflows.

---

### 📁 Document Hub
Organise documents with folder-based structure.  
Add dated records and upload/view attachments seamlessly.

---

### 📈 Profit & Loss (P&L)
Create structured P&L books with:
- Month-wise tracking  
- Lock/unlock functionality  
- Cash-out management  

---

### 🪙 Metals Ledger
Track precious metals transactions including:
- Gold  
- Silver  
- Platinum  

Upload attachments and maintain a detailed transaction history.

---

### 📦 FedEx Tracker
Import shipments and track:
- Payment status  
- Invoice summaries  
- Auto-sync updates  

---

## 🛠 Tech Stack

- Laravel (Backend)
- Blade + JavaScript (Frontend)
- MySQL (Database)
- Tailwind CSS (UI)
